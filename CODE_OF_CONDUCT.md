# Code of Conduct

## Our Standard

This project expects respectful, constructive collaboration.

Examples of positive behavior include:

1. Giving and receiving technical feedback in good faith.
2. Staying focused on facts and reproducible evidence.
3. Respecting differing viewpoints and experience levels.

Unacceptable behavior includes harassment, discrimination, personal attacks, or publishing private information without permission.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct and may remove comments, commits, issues, or contributors that violate these standards.

## Reporting

If you encounter unacceptable behavior, open a private GitHub security advisory or contact the maintainers through a private channel available on the repository profile.
